# 🚢 Titanic Survival Prediction

## Overview
A machine learning project developed using the Titanic dataset from Kaggle.

This project predicts passenger survival using classification models and data preprocessing techniques.


## Features
- Data preprocessing
- Missing value handling
- Feature engineering
- Classification model training
- Prediction evaluation


## Tech Stack
- Python
- Pandas
- NumPy
- scikit-learn
- Jupyter Notebook


## Machine Learning Workflow

### Data Cleaning
Handled missing values and prepared the dataset for model training.

### Feature Engineering
Processed categorical variables and optimized input features.

### Model Training
Applied machine learning classification algorithms for survival prediction.

### Evaluation
Measured prediction accuracy using validation metrics.


## Dataset
Titanic dataset from Kaggle.
